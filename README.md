# 手机云控系统

#### 项目介绍
- 本群控特点wifi连接手机，无需数据线
- 手机需要android5.0版本及以上，手机需要root。
- 手机通过云控系统连接服务器，手机类型：主控机、被控机
- 主控机对手机操作的手势转换成指令通过服务器发送给被控机，并实现复现
- 主控机对被控机进行批量APP的安装、升级、卸载
- 主控机录制手势操作脚本，在被控机上进行复现
